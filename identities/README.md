# Identity Namespaces

Per-agent doctrine and memory roots.

## Layout

- `main/`
- `researcher/`
- `huragok/`

Each contains:
- `SOUL.md`
- `USER.md`
- `HEARTBEAT.md`
- `MEMORY.md`
- `IDENTITY.md`
- `memory/`
- `workspace/` (optional scratch area)

Use `tools/identity/ensure-namespace.sh` to regenerate missing files safely.
