# 8-Week PPL Block v1

## Week 1

### Day 1: Push A
- Flat Bench Press: 5 x 8-10
- Incline Bench Press: 5 x 8-10
- Decline Bench Press: 5 x 10-12
- Shoulder Press: 5 x 8-10
- Lateral Raise: 3 x 12-15
- Triceps Extension: 3 x 10-15
- Dips: 2 x near failure optional, not mandatory

### Day 2: Run + Recovery
- Easy to moderate run: 30-45 min
- Mobility: 10-15 min
- Optional walk: 20-30 min later in the day

### Day 3: Pull A
- Barbell Deadlift: 5 x 5-8
- Barbell Bent Over Row: 5 x 8-10
- Neutral Grip Lat Pulldown: 5 x 8-10
- Barbell Bent Over Row: 3-4 x 8-10
- Barbell Biceps Curl: 3-4 x 10-12

### Day 4: Run + Recovery
- Easy to moderate run: 30-45 min
- Mobility: 10-15 min
- Optional walk: 20-30 min later in the day

### Day 5: Legs A
- Barbell Front Rack Split Squat: 5 x 8-10
- Barbell RDL: 4 x 8-10
- Step-Ups: 3 x 10-12 / side
- Lateral Lunge: 3 x 10-12 / side
- Standing Chop or ISO Split Squat Lift: 3 x 10-12 / side

### Day 6: Push B
- Incline Chest Fly: 5 x 10-12
- Middle Chest Fly: 5 x 10-12
- Decline Chest Fly: 5 x 10-12
- Shoulder Press: 3 x 8-10
- Lateral Raise: 3 x 12-15
- Triceps Extension: 3 x 10-15
- Dips: 2 x near failure optional, not mandatory

### Day 7: Full Recovery
- Easy walk optional
- Light mobility optional
- No lifting
- No hard conditioning
